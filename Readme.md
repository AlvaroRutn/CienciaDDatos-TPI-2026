## TPI
El trabajo consiste en la prediccion de demoras en vuelos, a partir del dataset "Vuelos.xlsx", para luego determinar los beneficios a otorgar a aquellos clientes cuyos vuelos hayan sido demorados, siempre a costo minimo.

## Recordatorios

    - Antes de ejecutar el codigo, recordar crear un "Venv" para la instalacion de las librerias requeridas para este trabajo.
    - Se poseen tres datasets:
        - Vuelos.xlsx Utilizado para el entrenamiento de los modelos de prediccion de demoras.
        - casosNuevos.xlsx Utilizado para evaluar el rendimiento del modelo ante nuevos casos y determinar la capacidad real de prediccion del modelo.
        - Clientes.xlsx Utilizado para la asignacion de beneficios a clientes demorados
